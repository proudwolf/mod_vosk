# vosk_mod

vosk_mod 是为freeswitch编译的一个模块，主要实现以下几个功能：

1、获取freeswtich的语音流

2、将freeswitch的语音流发送给ASR

3、获得ASR返回的语音识别文字

vosk_mod的具体代码解释，详情请参看如下博客的相关链接：
